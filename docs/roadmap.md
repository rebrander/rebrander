# Roadmap

## Phase 0 - Bootstrap

- Public GitHub repository.
- Go CLI skeleton.
- TypeScript frontend scaffold.
- Documentation set.
- GitHub Actions CI/CD.

## Phase 1 - Single Repository MVP

- Clone GitHub/GitLab repository.
- Build name graph from source and target URL.
- Dry-run plan.
- Structured config transforms for JSON/YAML/TOML.
- Path rename planner.
- Markdown reports.

## Phase 2 - Validation and Publishing

- Target repository creation.
- Branch push and PR/MR mode.
- Secret scan.
- License scan.
- Build/test command discovery.

## Phase 3 - Organization Rebranding

- GitHub organization and GitLab group discovery.
- Cross-repository dependency graph.
- Repository name mapping.
- Ordered migration plan.

## Phase 4 - SaaS

- OAuth GitHub/GitLab login.
- Job queue.
- Isolated workers.
- Report artifacts.
- Pages or hosted frontend at `rebrander.sh`.

## Phase 5 - AI-Assisted Rebranding

- Name classification.
- Asset replacement proposals.
- Binary inventory and rebuild decisions.
- Human review for risky transformations.
